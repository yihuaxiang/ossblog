# 爬山新装备

上周末刚爬[完香山](https://z.wiki/life/july-xiangshan.html)，这周对爬山的热情迟迟无法消退。

### 登山杖

趁着热情尚未退却，多买点装备多爬几次山。在京东上买了两个登山杖（一个开拓者、一个探路者）。

### 为什么

1. 登山杖在爬山过程中可以很好的缓解腿部压力（特别是负重的情况下）
2. 登山杖可以辅助保持身体平衡
3. 加快爬山速度
